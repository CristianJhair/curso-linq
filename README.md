# Curso-linq
Utilizar LINQ para bases de datos de Libros en formato JSON

## Requisitos
- Tener instalado NET 7.0
- Visual Studio de preferencia.

## Pasos para ejecutar proyecto
1. Clone el repositorio: `git clone https://github.com/CristianJhair/curso-linq`.
2. El archivo JSON de base de datos `books.json` se encuentra en la siguiente ruta (No mover): `bin\Debug\net7.0\books.json`
3. En `LinqQeries.cs` se encuentran las queries en formato LINQ.
4. En `Program.cs` descomente el reto de consulta LINQ que quiera visualizar `Reto 1), Reto 2) ...`
5. Ejecute el programa. Si está en VSC, ejecute en la terminal el comando `dotnet run`.
